# Financial-Stock-Analysis-Python-ML-GUI
This is a project about a financial prediction system application built within a Graphical User Interface (GUI) using Python. The application integrates concepts such as machine learning models with structural graph analytics, utilizing Matplotlib and the Tkinter library to create the GUI application window. 

The following application which was built is a financial Stock Market analyzer for various famous and well known companies.
